# Instructions for running both Apps

## Frontend <https://khansajeel.github.io/InformationRetrieval_2021_T34_Project>

```
- cd frontend
- npm install
- npm run start
```

## Backend

```
- cd backend
- virtualenv venv    ## Only first time
- source venv/bin/activate
- pip install -r requirements.txt
- python3 run.py
```
